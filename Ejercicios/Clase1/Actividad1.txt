Actividad #1Pregunta #1Si y no ya que internet y www no son lo mismo, internet es la infraestructura de red mientras que www es un servicio que corre sobre el internet.La distinction es importante ya que un sistema podria estar en internet sin ser parte de una www publica.Pregunta #2Si, se usan ambos ya que internet funciona como infraestructura de la red y la tecnología WWW ya que es la base de una aplicación web y sirve como el navegador. Sin embargo al estar restringida se comporta como una Intranet que es una red informativa privada habilitada solo para persona especificas, no como una web publica.Pregunta #3Ambos tienen distintas funciones, W3C define como se presenta al usuario la interfaz web, mientras que IETF define como se transfieren los datos (en este caso las notas) y que tan seguros están almacenados dentro del sistema. Tomando en cuenta cada funcionalidad lETF es la mas relevante, ya que lo mas importante del sistema será almacenar y manipular los datos de forma segura y estable.
